# Portafolio Profesional | Eduardo Hernández

Portafolio profesional desarrollado para mostrar mis habilidades, proyectos, certificaciones y experiencia como Ingeniero Informático enfocado en desarrollo web, sistemas empresariales y soluciones digitales modernas.

## Vista general

Este sitio fue diseñado con una interfaz moderna, responsiva y profesional. Presenta información sobre mi perfil, tecnologías, proyectos destacados, repositorios públicos de GitHub, cursos, certificaciones y medios de contacto.

El portafolio cuenta con un diseño oscuro, efectos visuales, tarjetas interactivas y una estructura adaptable para computadoras, tablets y dispositivos móviles.

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* Bootstrap
* Git
* GitHub
* Font Awesome

## Características principales

* Diseño responsivo para dispositivos móviles, tablets y computadoras.
* Sección de presentación profesional.
* Imagen de perfil personalizada.
* Sección de tecnologías utilizadas.
* Proyectos destacados con vista previa.
* Botones para acceder a Demo y repositorios de GitHub.
* Carga dinámica de repositorios públicos mediante la API de GitHub.
* Sección de cursos y certificaciones.
* Botones para visualizar certificados en formato PDF.
* Enlaces a GitHub, LinkedIn, WhatsApp y CV.
* Navegación interna mediante menú fijo.

## Cómo ejecutar el proyecto

1. Clona el repositorio:

```bash
git clone https://github.com/eherca0502/Portafolio.git
```

2. Abre la carpeta del proyecto en Visual Studio Code.

3. Abre el archivo `index.html` en tu navegador.

También puedes utilizar la extensión **Live Server** en Visual Studio Code para visualizar los cambios en tiempo real.
