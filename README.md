# Portafolio Profesional | Eduardo Hernández

Portafolio profesional desarrollado para mostrar mis habilidades, proyectos y experiencia como Ingeniero Informático enfocado en desarrollo web y soluciones digitales modernas.

## Vista general

Este sitio fue diseñado con un enfoque moderno, responsivo y profesional. Incluye información sobre mi perfil, tecnologías que utilizo, proyectos destacados y formas de contacto.

## Tecnologías utilizadas

- HTML5
- CSS3
- Bootstrap

## Secciones del portafolio

- Inicio
- Sobre mí
- Tecnologías
- Proyectos destacados
- Contacto
- Descarga de CV

## Proyectos incluidos

### Cyclonova

Sitio web corporativo desarrollado para una empresa real, enfocado en mejorar su presencia digital, experiencia de usuario y diseño moderno.

**Tecnologías:** PHP, MySQL y JavaScript.

### Sistema de Restaurante

Plataforma web para la gestión de pedidos, productos, clientes y administración de un negocio restaurantero.

**Tecnologías:** PHP, JavaScript y MySQL.

### Portafolio Personal

Sitio web responsivo creado para mostrar mi perfil profesional, habilidades técnicas y proyectos desarrollados.

**Tecnologías:** HTML, CSS y JavaScript.
